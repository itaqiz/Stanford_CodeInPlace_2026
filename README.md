# Stanford Code in Place 2026

Repository for my work, assignments, practice exercises, and final project completed during Stanford University's **Code in Place 2026** program.

## About the program

[Stanford Code in Place](https://codeinplace.stanford.edu/) is a global introductory programming course based on Stanford’s CS106A curriculum. The program teaches Python fundamentals through structured lessons, assignments, and live section support.

Topics covered include:

* Variables and data types
* Conditionals
* Loops
* Functions
* Lists and dictionaries
* Problem solving
* Basic graphics and program design
